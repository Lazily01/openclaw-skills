# Monitor Skill - 网页监控检查

## 触发场景

当用户说以下内容时，自动触发此 skill：
- "帮我监控这个网页"
- "检查这个网站能访问吗"
- "这个商品多少钱"
- "这个网页有 XX 关键词吗"
- "网页内容变了吗"
- "监控价格变化"

## 功能

网页监控与检查，支持：
- 🌐 **可用性检测** - 检查网站是否可访问、响应时间
- 💰 **价格提取** - 从网页提取商品价格
- 🔍 **关键词检测** - 检测关键词出现/消失
- 📄 **内容变化** - 检测网页内容是否变化
- 📸 **截图对比** - 可选，UI 变化检测

## 使用示例

### 可用性检测
```
用户：帮我检查 github.com 能访问吗
我：✅ 网站正常，响应时间 120ms
```

### 价格提取
```
用户：这个商品多少钱？https://item.jd.com/xxx
我：💰 当前价格：¥199
```

### 关键词检测
```
用户：这个网页有"降价"关键词吗？
我：🎯 发现关键词"降价"出现 2 次
```

### 内容变化检测
```
用户：帮我监控这个网页的内容变化
我：📄 内容哈希: abc123
    下次检查时对比哈希，变化则通知
```

### 综合检查
```
用户：帮我全面检查这个网站
我：生成检查报告：
    - 可用性: ✅
    - 响应时间: 150ms
    - 价格: ¥199
    - 关键词: 发现 3 个
```

## 监控模式

### 单次检查（Skill 模式）
按需调用，立即返回结果：
```
用户：检查这个网站
我：返回检查结果
```

### 持续监控（Agent 模式）
定时轮询，变化时报警：
```
用户：持续监控这个商品价格
我：已添加监控任务，价格变化时通知
```
