# Monitor Skill 使用说明

## 📁 文件结构

```
monitor-skill/
├── SKILL.md           # Skill 主说明文件
├── README.md          # 详细文档
├── skill.json         # 配置文件
├── 使用说明.md        # 本文件
└── scripts/
    └── monitor.py     # 核心脚本
```

## 🚀 快速开始

### 方式 1：通过 OpenClaw 使用（推荐）

1. 把 skill 放到 OpenClaw 目录：
```bash
cp -r ~/Desktop/monitor-skill ~/.openclaw/skills/
```

2. 对我说：
- "帮我检查这个网站能访问吗"
- "这个商品多少钱"
- "帮我监控这个网页内容变化"

### 方式 2：命令行使用

```bash
# 可用性检查
python3 scripts/monitor.py --url https://github.com --check availability

# 价格检查
python3 scripts/monitor.py --url https://item.jd.com/xxx --check price --threshold 100

# 关键词检测
python3 scripts/monitor.py --url https://example.com --check keyword --keywords "降价,促销"

# 内容变化检测
python3 scripts/monitor.py --url https://example.com --check content

# 全面检查
python3 scripts/monitor.py --url https://example.com --check all
```

## 📋 功能说明

### 1. 可用性检测

检查网站是否可访问：
- HTTP 状态码
- 响应时间
- 速度评级

```bash
python3 scripts/monitor.py --url https://github.com --check availability
```

### 2. 价格提取

从网页提取商品价格：
- 自动识别多种价格格式
- 可设置目标价格
- 价格达标时提醒

```bash
python3 scripts/monitor.py --url https://item.jd.com/xxx --check price --threshold 199
```

### 3. 关键词检测

检测网页是否包含关键词：
- 支持多个关键词
- 统计出现次数

```bash
python3 scripts/monitor.py --url https://example.com --check keyword --keywords "AI,GPT,ChatGPT"
```

### 4. 内容变化检测

追踪网页内容是否变化：
- 基于内容哈希对比
- 首次检查记录基准
- 后续检查对比变化

```bash
python3 scripts/monitor.py --url https://example.com --check content
```

## ⚙️ 配置

可选配置文件：`~/.monitor-skill/config.json`

```json
{
  "timeout": 30,
  "user_agent": "Mozilla/5.0 ..."
}
```

数据存储：`~/.monitor-skill/data/`
- 存储内容哈希
- 用于变化对比

## 📊 输出报告

使用 `--output` 参数保存报告：

```bash
python3 scripts/monitor.py --url https://example.com --check all --output report.md
```

## 注意事项

1. **登录限制**：某些网站需要登录才能访问
2. **价格提取**：依赖网页结构，不同网站效果不同
3. **内容变化**：只能检测文本内容变化
4. **频率限制**：避免过于频繁访问同一网站
