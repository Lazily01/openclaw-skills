# HR Skill 使用说明

## 📁 文件结构

```
hr-skill/
├── SKILL.md           # Skill 主说明文件
├── README.md          # 详细文档
├── skill.json         # 配置文件
├── 使用说明.md        # 本文件
├── scripts/
│   ├── resume_screener.py   # 核心脚本
│   └── email_reader.py      # 邮件读取模块
└── jobs/              # 岗位配置目录
    ├── 运营总监.json
    └── 软件测试工程师.json
```

## 🚀 快速开始

### 方式 1：通过 OpenClaw 使用（推荐）

1. 把 skill 放到 OpenClaw 目录：
```bash
cp -r ~/Desktop/hr-skill ~/.openclaw/skills/
```

2. 对我说：
- "帮我筛选这份简历"
- "从邮箱获取简历并筛选"

### 方式 2：命令行独立使用

1. 配置（首次）：
```bash
python3 scripts/resume_screener.py --setup
```

2. 添加岗位：
```bash
python3 scripts/resume_screener.py --add-job
```

3. 筛选简历：
```bash
# 本地文件
python3 scripts/resume_screener.py --file resume.pdf --job 运营总监

# 从邮箱
python3 scripts/resume_screener.py --from-email --job 运营总监
```

## 📬 邮箱配置

### 配置 IMAP（获取简历）

```bash
python3 scripts/resume_screener.py --setup
```

按提示输入：
- IMAP 邮箱地址
- IMAP 服务器（QQ: imap.qq.com, 163: imap.163.com）
- 授权码（不是邮箱密码！）

### 获取授权码

| 邮箱 | 获取方式 |
|------|----------|
| QQ 邮箱 | 设置 → 账户 → POP3/IMAP 服务 → 生成授权码 |
| 163 邮箱 | 设置 → POP3/SMTP/IMAP → 开启服务 → 获取授权码 |

## 📋 岗位配置

### 交互式添加
```bash
python3 scripts/resume_screener.py --add-job
```

### 手动创建
在 `jobs/` 目录下创建 JSON 文件：

```json
{
  "title": "运营总监",
  "required_experience": 3,
  "required_education": "本科",
  "required_skills": ["运营管理", "团队管理"],
  "bonus_skills": ["互联网经验"],
  "description": "岗位描述..."
}
```

## 🎯 匹配算法

| 项目 | 分值 |
|------|------|
| 经验达标 | 30 分 |
| 技能匹配 | 40 分 |
| 加分项 | 20 分 |
| 整体相关性 | 10 分 |

**推荐等级：**
- ⭐⭐⭐⭐⭐ 90%+ 强烈推荐
- ⭐⭐⭐⭐ 75-90% 推荐
- ⭐⭐⭐ 60-75% 可考虑
- ⭐⭐ 40-60% 不太匹配
- ⭐ <40% 不推荐

## ⚙️ 配置文件

位置：`~/.hr-skill/config.json`

```json
{
  "api_provider": "zai",
  "api_key": "your-api-key",
  "model": "glm-5",
  "email_sender": "your@163.com",
  "email_password": "授权码",
  "email_recipient": "recipient@example.com",
  "imap_server": "imap.qq.com",
  "imap_port": 993,
  "imap_email": "your@qq.com",
  "imap_password": "授权码"
}
```

## 🔒 安全提示

- 授权码不是邮箱密码，需要在邮箱设置中生成
- API Key 需要用户自行配置
- 配置文件保存在本地，请勿分享
