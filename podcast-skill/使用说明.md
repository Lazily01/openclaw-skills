# 播客 Skill 使用说明

## 📁 文件结构

```
podcast-skill/
├── SKILL.md           # Skill 主说明文件（OpenClaw 读取这个）
├── README.md          # 详细文档
├── skill.json         # 配置文件
├── 使用说明.md        # 本文件
└── scripts/
    └── generate_podcast.py   # 核心脚本
```

## 🚀 快速测试

### 方法 1：命令行测试

```bash
# 进入目录
cd ~/Desktop/podcast-skill

# 测试生成音频（不需要邮箱配置）
python3 scripts/generate_podcast.py --text "你好，这是一段测试播客内容。"

# 指定男声
python3 scripts/generate_podcast.py --text "测试内容" --voice yunyang
```

### 方法 2：配置邮箱后发送邮件

```bash
# 先配置邮箱
python3 scripts/generate_podcast.py --setup

# 然后发送
python3 scripts/generate_podcast.py --text "测试内容" --email
```

### 方法 3：在 OpenClaw 中使用

把这个 skill 放到 OpenClaw 的 skills 目录：
```bash
cp -r ~/Desktop/podcast-skill ~/.openclaw/skills/
```

然后重启 OpenClaw 或刷新 skills。

之后直接对我说：
- "把这段话生成播客"
- "用男声朗读"

## 📧 邮箱配置

**重要：邮箱信息需要用户自己配置，不提供默认值！**

### 方式 1：交互式配置
```bash
python3 scripts/generate_podcast.py --setup
```

### 方式 2：手动编辑配置文件
配置文件位置：`~/.podcast-skill/config.json`

```json
{
  "email_sender": "your@163.com",
  "email_password": "授权码（不是邮箱密码）",
  "email_recipient": "recipient@example.com",
  "smtp_server": "smtp.163.com",
  "smtp_port": 465,
  "default_voice": "xiaoxiao"
}
```

### 方式 3：命令行参数
```bash
python3 scripts/generate_podcast.py \
  --text "内容" \
  --email \
  --recipient user@example.com \
  --sender your@163.com \
  --password 授权码
```

pip3 install edge-tts --break-system-packages

## 🔒 安全提示

- 授权码不是邮箱密码，需要在邮箱设置中生成
- 配置文件保存在本地，请勿分享
- 支持主流邮箱：163、QQ、Gmail、Outlook 等
